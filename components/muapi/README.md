# MuAPI

[MuAPI](https://muapi.ai) is a generative media API aggregator giving you access to 50+ AI models for image generation, video generation, audio, and 3D through a single unified API.

## Actions

- **Generate Image** — Create images from text using Flux, Midjourney, GPT-4o, Imagen 4, Seedream, and more
- **Generate Video** — Generate videos from text using Veo3, Veo4, Kling, Wan, Seedance, Runway, Sora, and more
- **Get Prediction Result** — Check the status and retrieve output of any generation job

## Authentication

Get your API key from [muapi.ai/dashboard/api-keys](https://muapi.ai/dashboard/api-keys).
