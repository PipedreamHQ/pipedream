# SendIt Components

[SendIt](https://sendit.infiniteappsai.com) is an AI-native social media management tool that enables publishing to multiple platforms from a single interface.

## Supported Platforms

- LinkedIn
- Instagram (Business/Creator accounts)
- Threads
- TikTok
- X (Twitter)

## Authentication

SendIt uses API Key authentication. Get your API key from the [SendIt Dashboard](https://sendit.infiniteappsai.com/dashboard).
