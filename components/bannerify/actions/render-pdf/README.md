# Overview

Render PDF generates a PDF from a Bannerify template and returns a hosted file URL.

# Example Use Cases

- Generate invoices from order events.
- Create labels or certificates from form submissions.
- Build personalized PDF documents in automation workflows.

# Getting Started

Select a Bannerify account, enter the template ID, and pass optional modifications as JSON.

# Troubleshooting

If the step fails, confirm the template ID belongs to the connected Bannerify project and the modifications JSON is valid.
