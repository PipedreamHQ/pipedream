# Overview

Render Image generates an image from a Bannerify template and returns a hosted file URL.

# Example Use Cases

- Generate social images from form submissions.
- Create ecommerce product banners from catalog data.
- Render personalized campaign assets from workflow variables.

# Getting Started

Select a Bannerify account, enter the template ID, choose `png`, `jpeg`, or `webp`, and pass optional modifications as JSON.

# Troubleshooting

If the step fails, confirm the template ID belongs to the connected Bannerify project and the modifications JSON is valid.
