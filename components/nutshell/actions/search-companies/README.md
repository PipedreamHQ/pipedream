# Overview

Searches Nutshell companies (accounts) by string. Returns raw API results as an array of account stubs.

# Example Use Cases

- Find companies by name before creating leads or contacts
- Look up company IDs for use in downstream steps

# Getting Started

Connect your Nutshell account in the step. See the [Nutshell app README](../../README.md) for account setup.

# Troubleshooting

See the [Nutshell app README](../../README.md) for authentication and API troubleshooting.
