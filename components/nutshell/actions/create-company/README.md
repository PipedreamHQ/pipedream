# Overview

Creates a new company (account) in Nutshell. You can set name, addresses, emails, phones, URLs, industry, territory, and custom fields.

# Example Use Cases

- Create Nutshell companies from form submissions or other CRMs
- Add new accounts when they appear in a spreadsheet or database
