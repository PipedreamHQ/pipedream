# Overview

Searches Nutshell leads by string (matches lead names, descriptions, etc.). Returns formatted results including id, description, status, completion, value, primaryCompanyName, primaryContactName, isOverdue, lastContactDate, and dueTime.

# Example Use Cases

- Find leads matching a search term for reporting or updates
- Look up leads before updating status or adding notes
