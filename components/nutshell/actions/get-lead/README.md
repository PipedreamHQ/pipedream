# Overview

Retrieves a single lead from Nutshell by ID or by lead number (e.g. 1000 for Lead-1000). Returns formatted lead data including description, status, primary company and contact, and custom fields.

# Example Use Cases

- Fetch full lead details before updating or closing the lead
- Enrich workflow data with lead information from Nutshell

# Getting Started

Connect your Nutshell account in the step. See the [Nutshell app README](../../README.md) for account setup.
