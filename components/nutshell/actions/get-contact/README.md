# Overview

Retrieves a single contact from Nutshell by ID. Returns formatted contact data including name, emails, phones, and associated companies.

# Example Use Cases

- Look up contact details before updating them in another step
- Enrich workflow data with full contact information from Nutshell

# Getting Started

Connect your Nutshell account in the step. See the [Nutshell app README](../../README.md) for account setup.

See the [Nutshell app README](../../README.md) for authentication and API troubleshooting.
