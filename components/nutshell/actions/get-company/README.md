# Overview

Retrieves a single company (account) from Nutshell by ID. Returns full account data including name, contacts, addresses, and custom fields.

# Example Use Cases

- Look up company details before updating them or creating leads
- Enrich workflow data with company information from Nutshell

# Getting Started

Connect your Nutshell account in the step. See the [Nutshell app README](../../README.md) for account setup.
