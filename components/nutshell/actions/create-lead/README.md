# Overview

Creates a new lead in Nutshell. You can set description, status, tags, associated companies and contacts, and custom fields.

# Example Use Cases

- Create Nutshell leads from form submissions, landing pages, or other CRMs
- Add new leads when they appear in a spreadsheet or database
