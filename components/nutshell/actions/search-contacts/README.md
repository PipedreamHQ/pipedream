# Overview

Searches Nutshell contacts by string (matches names, emails, etc.). Returns an array of contacts with id, displayName, primaryEmail, and primaryPhone.

# Example Use Cases

- Find contacts matching a search term for follow-up workflows
- Look up contacts before creating or updating leads

# Getting Started

Connect your Nutshell account in the step. See the [Nutshell app README](../../README.md) for account setup.
