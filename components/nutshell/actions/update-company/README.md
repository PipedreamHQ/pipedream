# Overview

Updates an existing company (account) in Nutshell. Only provided fields are updated; custom fields from your Nutshell pipeline can be set via props or passed from a previous step.

# Example Use Cases

- Sync company data from another app into Nutshell
- Update account details when information changes in a workflow

# Getting Started

Connect your Nutshell account in the step. See the [Nutshell app README](../../README.md) for account setup.
