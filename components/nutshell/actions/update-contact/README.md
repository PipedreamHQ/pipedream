# Overview

Updates an existing contact in Nutshell. Only provided fields are updated; custom fields from your Nutshell pipeline can be set via props or passed from a previous step.

# Example Use Cases

- Sync contact data from another app (e.g. CRM, form) into Nutshell
- Update contact details when information changes in a workflow

# Getting Started

Connect your Nutshell account in the step. See the [Nutshell app README](../../README.md) for account setup.
