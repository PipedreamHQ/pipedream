# Overview

Emit events when a new person is created in Nutshell. This is a polling source that checks for new people on a configurable interval.


# Example Use Cases

- Trigger workflows when new people are added to Nutshell (e.g. sync to CRM, send welcome emails)
- Automate follow-up when a new contact is created

# Getting Started

Connect your Nutshell account in the step. See the [Nutshell app README](../../README.md) for account setup.
