# Overview

Emit new events when a new lead is created in Nutshell. This is a polling source that checks for new leads on a configurable interval.

# Example Use Cases

- Trigger workflows when new leads are created (e.g. sync to Google Sheets, send Slack notifications)
- Automate follow-up when a new lead enters your pipeline

# Getting Started

Connect your Nutshell account in the step. See the [Nutshell app README](../../README.md) for account setup.
