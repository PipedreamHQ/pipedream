# Overview

Emit new events when a lead is won in Nutshell. This is a polling source that checks for newly won leads on a configurable interval.

# Example Use Cases

- Trigger workflows when deals are closed (e.g. send congratulations, update reporting)
- Automate post-sale steps when a lead is marked as Won

# Getting Started

Connect your Nutshell account in the step. See the [Nutshell app README](../../README.md) for account setup.
