# SendIt Components

[SendIt](https://sendit.infiniteappsai.com) is an AI-native social media management tool that enables publishing to multiple platforms from a single interface.

## Supported Platforms

- LinkedIn
- Instagram (Business/Creator accounts)
- Threads
- TikTok
- X (Twitter)

## Actions

| Action | Description |
|--------|-------------|
| **Publish Post** | Publish content to social media platforms immediately |
| **Schedule Post** | Schedule content for future publication |
| **Cancel Scheduled Post** | Cancel a pending scheduled post |
| **Trigger Scheduled Post** | Publish a scheduled post immediately |
| **Validate Content** | Validate content against platform requirements |
| **List Accounts** | List connected social media accounts |
| **List Scheduled Posts** | List pending scheduled posts |

## Sources (Triggers)

| Source | Description |
|--------|-------------|
| **Post Published** | Triggers when a post is successfully published |
| **Post Scheduled** | Triggers when a post is scheduled |
| **Post Failed** | Triggers when a post fails to publish |

## Authentication

SendIt uses API Key authentication. Get your API key from the [SendIt Dashboard](https://sendit.infiniteappsai.com/dashboard).
