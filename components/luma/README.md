# Overview

The Luma API lets you manage events, guests, invitations, and tickets for a Luma calendar. With Pipedream, you can automate event operations such as syncing upcoming events, pulling guest lists into a CRM, sending notifications, and registering guests from forms or other systems.

# Example Use Cases

- **Sync Events to a CRM**: List upcoming Luma events and create or update matching campaign records in a CRM.
- **Guest List Reporting**: Pull all guests for an event and append the data to a spreadsheet for attendance tracking.
- **Automated Invitations**: Trigger Luma invites when a lead reaches a specific stage in another system.
