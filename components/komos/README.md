# Komos

Komos turns browser workflows into reusable automations with API triggers, schedules, credentials, logs, and human review.

This Pipedream component package includes:

| Action | Purpose |
| --- | --- |
| `queue-task-run` | Queue a saved Komos task with optional inputs and an idempotency key. |
| `get-task-run` | Fetch the latest status and outputs for a Komos task run. |
