# Polar

Polar is a monetization platform for developers and creators. This integration lets you automate subscriptions, orders, webhooks, and more using the [Polar API](https://polar.sh/docs/api-reference).

## Setup

Connect your Polar account using an OAuth access token. You'll also need your **Organization ID**, which you can find in your [organization settings](https://polar.sh/dashboard) under **Settings → General → Identifier**.
